HTML & CSS Cheatsheet

Elements & Structure
HTML Tag Name Description
a Anchor Used to create hyperlinks in an HTML document
body Body Represents the content rendered on the web browsers
br Line Break Creates a line break in text
div Division Used as a container that divides an HTML document into sections
em Emphasis Emphasizes text and browsers will usually italicize the emphasized text by default
h1-h6 Heading Six different level of heading elements
head Head Contains general information about an HTML page that isn't displayed on the page itself
html HTML The root of an HTML document, added after the !DOCTYPE declaration
img Image Embeds images in documents
li List Item Creates list items inside an ordered or unordered list
ol Ordered List Creates a list of items in sequential order
p Paragraph Contains and displays a block of text
span Span An inline container for text
strong Strong Highlights important, serious, or urgent text in bold
title Title Contains a text that defines the title of an HTML document
ul Unordered List Creates a list of items in no particular order
video Video Embeds a media player for video playback
Tables
HTML Tag Name Description
table Table Used to represent two-dimenstional table made of rows and columns
tbody Table Body Contains all table data other than table heading and table footer content
td Table Data Can be nested inside a table row element to add a cell of data to a table
th Table Heading Used to add titles to rows and columns of a table
tfoot Table Footer Uses table rows to give footer content or to summarize content at the end of a table
thead Table Head Defines the headings of table columns encapsulated in table rows
tr Table Row Used to add rows to a table before adding table data and headings
Forms
HTML Tag Name Description
datalist Data List Used to store a list of options
form Form Used to collect and send information to an external source
input Input Used to render a variety of input fields on a webpage
label Label Provides identification for a specific input
select Select Can be used to create a dropdown list
textarea Text Area Used when creating a text-box for multi-line input
Visual Rules
CSS Property Name Description
background-color Background Color Controls the background color of elements
background-image Background Image Sets the background image of an element
color Color Controls the foreground text color of an element
font-family Font Family Used to specify the typeface in a rule set
font-size Font Size Used to set text sizes
font-weight Font Weight Used to set the weight or boldness of text
!important Important Used on declarations to override any other declaration for a property
opacity Opacity Used to control the transparency of an element
text-align Text Align Used to set the text alignment of inline contents
Display & Positioning
CSS Property Name Description
clear Clear Specifies how an element should behave when it bumps into another element
display Display Determines the type of render block for an element
float Float Determines how far left or how far right an element should float within its parent
position Position Provides options for positioning elements on a webpage
z-index Z-Index Specifies how far back or how far forward an element will appear
