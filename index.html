HTML & CSS Cheatsheet

Elements

Tag Name Description
a Anchor Used to create hyperlinks in an HTML document
body Body Represents the content rendered on the web browsers
br Line Break Creates a line break in text
div Division Used as a container that divides an HTML document into sections
em Emphasis Emphasizes text and browsers will usually italicize the emphasized text by default
h1-h6 Heading Six different level of heading elements
head Head Contains general information about an HTML page that isn't displayed on the page itself
html HTML The root of an HTML document, added after the !DOCTYPE declaration
img Image Embeds images in documents
li List Item Creates list items inside an ordered or unordered list
ol Ordered List Creates a list of items in sequential order
p Paragraph Contains and displays a block of text
span Span An inline container for text
strong Strong Highlights important, serious, or urgent text in bold
title Title Contains a text that defines the title of an HTML document
ul Unordered List Creates a list of items in no particular order
video Video Embeds a media player for video playback
